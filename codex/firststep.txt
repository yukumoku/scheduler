あなたは熟練したLaravel + React + TypeScriptエンジニアです。

これから「ScheduleCraft」というイベント向けスケジュール・シフト自動作成Webアプリを実装します。

まずはMVPの基盤を作成してください。

## 技術構成

* Laravel 13.x
* PHP 8.3以上
* PostgreSQL
* Laravel Sanctum
* Laravel Socialite
* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui
* TanStack Query
* React Hook Form
* Zod

## アプリ概要

ScheduleCraftは、文化祭・体育祭・部活動・サークル・ボランティア・地域イベントなどで使える、汎用的なスケジュール・シフト自動作成サービスです。

管理者がグループとイベントを作成し、メンバーを招待します。
メンバーは参加可能日時を入力します。
管理者は希望提出状況をもとに、公平性や希望反映率を考慮してシフトを自動生成します。
完成したシフトは公開・Excel出力できます。

## 実装ルール

* TypeScriptではanyをなるべく使わない
* Laravel Controllerにビジネスロジックを書きすぎない
* 将来的にService / Actionへ分離しやすい構成にする
* APIレスポンスは以下の形に統一する

```json
{
  "success": true,
  "data": {},
  "error": null
}
```

エラー時：

```json
{
  "success": false,
  "data": null,
  "error": {
    "code": "ERROR_CODE",
    "message": "message"
  }
}
```

* UI文言は日本語
* コード内コメントは英語
* DBカラムはsnake_case
* APIレスポンスはcamelCase
* まずは動くMVPの土台を優先する
* 完全なシフト生成アルゴリズムはまだ実装しなくてよい
* Excel出力もまだ実装しなくてよい
* ただし後から追加しやすい構成にする
